# Beauty Product Safety Explorer

## Project Description
Beauty Product Safety Explorer is a web application that allows users to search and explore cosmetic adverse event reports sourced from the openFDA Cosmetic Adverse Events API. Users can look up beauty and skincare product safety information, see reported reactions, and better understand the risks associated with products.

## Target Browsers
- Google Chrome
- Safari
- Microsoft Edge
